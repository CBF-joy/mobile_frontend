# KBSC mobile_frontend

### 환경설정
react-native -v : 0.63.2<br>
react-native-cli : 2.0.1<br>
expo --version : 4.1.6<br>

<br><br>
### 실행방법
### 1. development server 실행
- cd mobile_frontend
- expo start
- 📌 DevTools : http://localhost:19002 실행
- 📌 client : http://localhost:19000 실행

### 2. expo 클라이언트 실행
- expo 앱 다운로드 및 실행
- QR코드 스캔 후 접속
